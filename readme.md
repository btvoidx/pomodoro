# Web pomodoro timer
> Made to procrastinate without guilt on my uni exams

# Tech stack
- [Alpine.js](https://alpinejs.dev) for reactivity
- [tailwindcss](https://tailwindcss.com) for styling
- [Github Pages](https://pages.github.com) for hosting
